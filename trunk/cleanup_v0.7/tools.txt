# [*] Tool categories:
# - java: Java-based tool. Requires the Java Runtime Environment (JRE) to run.
# - CLI:  Command-line tool.
# - GUI:  Graphical (X-windows) tool.
# - DIR:  Tool that only contains data & files within its directory (no exec).
# - OS:   Tool available on the Operating System official repositories.
#         Eg. apt-get and "dpkg --list"
# - OS (tgz): Tool installed in the OS layout from its official package.
# - web:  Web-server tool installed under the Apache document root (/var/www).
#
#
# All tools have a Gnome menu file under the Applications menu:
# - Except tools in the OS and web categories
# - For java & GUI & DIR tools, it points to its launch script
# - For CLI tools it points to its gnome-terminal profile
# 
# All web tools have a link in the Firefox bookmarks
#
# All tools have a launch srcipt under /home/samurai/tools, except OS tools
#
# CLI tools have a gnome-terminal profile (except w3af):
# - The tool profile points to its launch script
#
# TGZ tools can be invoked from any terminal. 
# - A specific launch script is not created for them.
#
